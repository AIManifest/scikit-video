.. _utilities:

===========================
Utility functions
===========================

.. currentmodule:: skvideo.utils

`skvideo.utils` provides several utility functions for manipulating video data.

Standardized shapes
-----------------------

Use `skvideo.utils.vshape` to reshape video data (or image data) into a standard shape.

Grayscaling
-----------------------

Use `skvideo.utils.rgb2gray` to convert video data into its grayscaled version.
